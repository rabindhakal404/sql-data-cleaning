# 🧹 SQL Data Cleaning Project

## 🔍 Overview

This project focuses on cleaning and preprocessing raw datasets using SQL. The goal is to transform messy, inconsistent data into a structured and analysis-ready format.

## 🎯 Objective

To demonstrate SQL skills in data cleaning, including handling missing values, duplicates, inconsistent formats, and improving overall data quality.

## 🛠 Tools Used

* SQL (MySQL)
* Database Management System

## ⚙️ Key Data Cleaning Operations

* Handling missing values (NULLs)
* Removing duplicate records
* Standardizing text and categorical values
* Fixing inconsistent data formats
* Data type corrections
* Filtering and validating records

## 📂 Project Structure

```
sql-data-cleaning/
│── DataCleaningProject.sql     # Main SQL script for cleaning
│── layoffs.csv                 # Raw dataset 
│── cleaned_output.csv          # Cleaned dataset 
│── README.md                   # Project documentation
```

## 🧠 What I Learned

* Writing efficient SQL queries for data preprocessing
* Understanding real-world data quality issues
* Preparing datasets for analysis and visualization

## 🚀 Outcome

A clean and structured dataset ready for exploratory data analysis (EDA) and business reporting.
